1. Descargar el proyecto desde el repositorio de GitHub https://github.com/SergioBarberoBascones/ShoppingList.git
2. Ejecutar el archivo ShoppingList_WIN.bat ó ShoppingList_LINUX.sh dentro de artifacts/ShoppingList_jar según el sistema operativo que tengas.
3. Tener en cuenta que los archivos autogenerados se crean en la misma carpeta que los ejecutables.
